windirs
---------
A ruby gem to convert between Unix, Windows, and Cygwin paths.

Installation
---------
`gem install windirs`

or, if you want the latest and the greatest, 

    git clone https://github.com/windirs
    cd windirs
    rake install

(use `sudo` as necessary)

Runtime Requirements
---------
ruby >= 1.9.2

Build Requirements
---------
rake
